# solving_equations
Данное веб-приложение создает тест для решения квадратных уравнений. В нем вы можете тренировать свои знания!
В программе пользователь проходит регистрацию (Вводит имя и возраст). Далее выбирает тип уравнений (приведенные или нет) и их количество. Далее переходит к тесту, решает уравнения и нажимает кнопку "проверить". Также на странице теста указана информация об условиях ввода ответа. На странице результатов отображаются ответы пользователя и верные ответы на уравнения. Ответы сравниваются и подсчитывается количество правильно решенных уравнений, выставляется оценка. Статистика пользователей будет сохранятся.
